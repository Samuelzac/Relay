FAQ
Do viewers need to log in?

No. Anyone with the watch link can view (subject to the concurrent viewer limit).

Do I need an app to broadcast?

No app install. Broadcasting is done in a mobile browser via a private broadcast link.

Is the stream recorded?

Not in this MVP. Relay is focused on simple live delivery. Recording can be added later.

Hosts are responsible for content. See the Terms for prohibited content and enforcement.
